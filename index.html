API 速率限制不合理

人类导向的限流策略严重制约 Agent 高频交易能力

延迟竞争不对等

MEV 机器人和机构享有基础设施优势，普通 Agent 处于劣势

缺乏专属激励机制

没有针对 Agent 做市、流动性提供的专项奖励

反操纵机制缺失

API 示例 — gRPC 下单

place_order.proto 
message PlaceOrderRequest {
  string agent_id = 1;
  string pair = "BTC/USDT";
  OrderSide side = 2;
  OrderType type = 3;
  string price = "50000.00";
  string quantity = "1.5";
  TimeInForce tif = 4;
  // AI 专属参数
  double aggressiveness = 5;  // 0-1 激进程度
  repeated Condition conditions = 6;  // 条件链
}

service TradingService {
  rpc PlaceOrder(PlaceOrderRequest) returns (OrderResponse);
  rpc PlaceBatchOrders(BatchRequest) returns (BatchResponse);
  rpc StreamOrderUpdates(stream OrderRequest) returns (stream OrderUpdate);
}
